# CS 4780: Introduction to Machine Learning

**Spring 2024** | University Name

## Course Description

A comprehensive introduction to machine learning covering supervised learning, unsupervised learning, and modern deep learning techniques. Includes hands-on programming assignments and a final project.

## Topics Covered

- Linear regression and classification fundamentals
- Decision trees, SVMs, and ensemble methods
- Neural networks and deep learning
- Unsupervised learning and dimensionality reduction
- Fairness, ethics, and practical ML considerations

## Assessment

- Programming Assignments: 40%
- Midterm Exam: 15%
- Final Exam: 20%
- Final Project: 20%
- Participation: 5%
