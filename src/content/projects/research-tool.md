# VisualizeML: Interactive ML Visualization Toolkit

A comprehensive Python library for creating interactive, publication-quality visualizations of machine learning models, training dynamics, and high-dimensional data with a focus on interpretability.

## Features

- Model architecture and computation graph visualization
- Training dynamics and learning curves
- Decision boundary and prediction surface plots
- Feature importance and attention maps
- High-dimensional embedding visualization (t-SNE, UMAP)
- Publication-quality output (PNG, SVG, PDF)

## Installation

```bash
pip install visualizeml
```
